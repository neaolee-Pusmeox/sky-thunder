# SKY THUNDER · 雷电风暴

**一款打开就能玩的竖版弹幕射击游戏。** 没有安装包，没有加载条，双击 HTML 立刻上天。

你驾驶「雷霆」系列战机，穿越 7 大空域，在弹幕里杀出一条血路，最终对决 **湮灭之神**。全程纯 Canvas 绘制、Web Audio 合成音效，整个游戏只有一个文件。

---

## 为什么值得玩一把

- **7 关完整战役**：每关 5 波敌潮 + 1 名独特 Boss，从装甲列车一直打到终焉神殿。
- **5 种武器，6 级进化**：散弹、穿透激光、追踪导弹、雷电链……吃到 W 道具还能解锁 Lv.6 终极形态。
- **三档难度，三种活法**：想休闲选 EASY（自动炸弹保命）；想冲榜选 HARD（自杀弹 + 高分倍率）。
- **STG 老味道**：擦弹加分、连击 COMBO、TENSION 动态加压，越打越疯。
- **手机也能玩**：单指拖动跟手开火，右侧一键暂停、切武器、放炸弹。

---

## 30 秒上手

1. 用浏览器打开 [`raiden-game.html`](raiden-game.html)
2. 选难度，点 **出击**
3. 键盘：`WASD` 移动，`J` 射击，`K` 炸弹，`M` 切武器，`P` 暂停
4. 手机：单指拖动；右侧按钮操作；双指按住后松手也可放炸弹

菜单里的 **? 游戏说明** 有完整手册。
